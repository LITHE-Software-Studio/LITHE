# LITHE.

## Project workflow  
```
npm install  
npm run build  
npm run start  
```

## Directory
1. client side: client folder  
- src/api -> implement calls to server  
- src/components -> build front end of the website  
- src/utilities -> helper files/functions  
2. server side: server folder
